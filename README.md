# True-Cost-Calculator
🌍 True Cost Calculator is an Agentic AI system that helps users estimate the real cost of international purchases. It analyzes product price, import duty, shipping charges, and currency conversion to calculate the final cost. A simple Streamlit interface is used for user interaction.

Objective
💡 Estimate the true cost of international products
🌐 Include tax, duty, shipping, and currency conversion
📊 Help users make better purchasing decisions

Features
🤖 Agent-based cost calculation
💱 Currency conversion support
🧾 Import duty and tax estimation
📊 Simple Streamlit user interface

Technologies Used
🐍 Python
⚡ FastAPI
🤖 Agentic AI (LangChain)
🗄️ SQLite
🎨 Streamlit

How It Works
1️⃣ User enters product price and country
2️⃣ AI agent processes cost factors
3️⃣ Taxes, duty, shipping, and currency conversion are calculated
4️⃣ Final true cost is displayed
